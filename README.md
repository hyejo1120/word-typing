# word-typing

영어 단어 공부용 타이핑 연습 게임 🎮⌨️

단어가 화면 **아래에서 위로** 올라옵니다. 위로 사라지기 전에 **영어 스펠링**을
정확히 입력하세요! 각 단어에는 **한국어 뜻**이 함께 표시됩니다.

## 실행 방법

별도 설치가 필요 없습니다. `index.html` 파일을 웹 브라우저에서 열기만 하면 됩니다.

```bash
# 예시 (macOS)
open index.html

# 또는 그냥 탐색기/파인더에서 index.html 더블클릭
```

## 게임 방법

- 화면 아래에서 영어 단어가 한국어 뜻과 함께 떠오릅니다.
- 입력창에 영어 스펠링을 타이핑하면, 첫 글자가 맞는 단어가 자동으로 선택됩니다.
- 단어를 끝까지 정확히 입력하면 사라지고 점수를 얻습니다.
- 단어가 화면 위로 넘어가 사라지면 **생명(❤)** 이 하나 줄어듭니다.
- 생명을 모두 잃으면 게임이 끝납니다.

## 특징

- **레벨 시스템**: 단어 8개를 맞출 때마다 레벨이 오르고 속도가 빨라집니다.
- **정확도 측정**: 타이핑 정확도를 실시간으로 보여줍니다.
- **일시정지 / 다시 시작** 지원 (`ESC` 키로 입력 초기화).

## 단어 추가하기

`index.html` 안의 `WORDS` 배열에 `["영어단어", "한국어뜻"]` 형식으로 추가하면 됩니다.

```js
const WORDS = [
  ["apple", "사과"],
  ["book", "책"],
  // 여기에 추가...
];
```
